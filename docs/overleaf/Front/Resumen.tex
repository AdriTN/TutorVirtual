\large{
En este Trabajo Fin de Grado se diseña e implementa un \emph{tutor virtual
personal} que emplea inteligencia artificial para ofrecer aprendizaje
individualizado en distintas materias.  
Un panel de administración permite configurar cursos, asignaturas y temarios,
mientras que el sistema ajusta de forma dinámica la dificultad de los ejercicios
según el nivel declarado de cada estudiante.  
La plataforma ofrece una interfaz web intuitiva y paneles de progreso que
facilitan la trazabilidad del aprendizaje.

La arquitectura \textit{backend} se apoya en microservicios FastAPI (Python),
un pipeline de \emph{Retrieval Augmented Generation} que combina una base
vectorial con un modelo grande de lenguaje desplegado en Ollama para generar
retroalimentación contextualizada, y un esquema de métricas almacenadas en
PostgreSQL. Todo ello se expone mediante una API REST protegida con JWT.
}
